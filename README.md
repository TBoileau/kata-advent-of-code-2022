# Advent Of Code 2022

## Installation
```
git clone git@github.com:TBoileau/kata-advent-of-code-2022.git
cd kata-advent-of-code-2022
composer install
```

## Execution
```
php bin/aoc DAY PART
```

Exemple :
```
php bin/aoc 1 1
```

## Tests
```
vendor/bin/phpunit --testdox
```
